I work on systems of nonlinear differential equations. My thesis focuses on computing Input/Ouput integral equations from a dynamical system using integral elimination. The aim is to perform parameter estimation from the integral equations to deal with noisy measurements. Another part of my work is using deep learning techniques to improve the integral elimination algorithm I am working on.
Keywords: Computer Algebra, Computer Science, integral elimination, deep learning, NLP, Transformer

Software

  • IntegralElimination: Algorithm for performing integral elimination for nonlinear integral equations
  • parameter-estimation-IO-equations: Package for computing cost/gradient/hessian functions from an integro-differential equation and to do parameter estimation.
